# AI-Based Turkish License Plate Recognition System

## Overview

AI-powered Turkish License Plate Recognition (LPR) system built using YOLO11 and PaddleOCR.

Features:

- Turkish License Plate Detection
- Real-Time Video Processing
- PaddleOCR Integration
- Duplicate Plate Filtering
- Vehicle Tracking
- CSV Logging
- Dashboard & Reporting
